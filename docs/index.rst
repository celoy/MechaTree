MechaTree
=========

MechaTree is a Python (with Cython + C++) port of the Fortran90 tree-growth
simulator from Eloy et al., *Nat Commun* 8:1014 (2017). It models tree
architecture under biomechanical (wind loads, self-supporting limits) and
photosynthetic (competition for light) constraints, producing self-pruning,
allometric scaling, and self-similar branching.
